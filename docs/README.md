# Docs

This directory should contain human-facing documentation for the MJLab port.

Expected future files:

- `architecture.md`: environment, model, algorithm, and export architecture.
- `setup.md`: dependency installation and MJLab setup.
- `training.md`: training commands, config explanations, and expected metrics.
- `evaluation.md`: evaluation procedure and metrics.

Docs should explain decisions and project workflow.
