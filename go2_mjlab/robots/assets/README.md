# Robot Assets

This directory should contain MJLab/MuJoCo assets for the Unitree Go2.

Expected future files:

- `go2.xml`: main MJCF model file used by MJLab.
- `meshes/`: mesh files referenced by the MJCF.
- `textures/`: texture files referenced by the MJCF.

The MJCF should define the robot bodies, joints, actuators, inertial properties, collision geoms, visual geoms, and any asset references needed by the simulator.
